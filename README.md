# True Wellness Movement
**Heavenly Herb Garden & Children’s Wellness Center**

A community-led, nature-based nonprofit initiative focused on building an outdoor wellness space where children learn mindfulness, movement, and emotional balance through nature-based play.

---

## 🚀 Site Features
- **Nature-First Design:** Immersive, high-impact imagery.
- **Conversion-Focused:** Integrated DonorBox for fundraising and MailerLite for community engagement.
- **Modern Performance:** Optimized for fast loading with lazy-loaded images and minimalist CSS.
